## Проектная работа

# Тема: Система автоматизации работы библиотеки, учёта книг, пользователей, выдачи книг пользователям

### Проект выполнен с использованием фреймворка Django. База данных для демонстрации работы включена в репозиторий.

### Установка зависимостей осуществляется командой:
*pip install -r requirements.txt*


### Запуск сервера производится командой:
*python manage.py runserver*

### Запуск тестов
*python manage.py test*

Выполняются проверки моделей:

*CustomUser* Создание, суперпользователь, обязательные поля

*Author* Валидация дат, строковое представление

*Publisher* Уникальность названия

*Book* Год издания (не в будущем, ≥1), строковое представление

*Cart* Только для обычных пользователей, OneToOne

*BorrowedBook* Состояние выдачи/возврата
